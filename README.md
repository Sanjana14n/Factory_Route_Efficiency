# 📖 Project Overview

This project analyzes the shipping performance of **Nassau Candy Distributor** by evaluating factory-to-customer delivery routes across multiple regions of the United States.

The objective is to identify shipping bottlenecks, compare logistics performance across factories and shipping methods, evaluate route efficiency, and provide business recommendations through an interactive Streamlit dashboard.

The project transforms raw shipment data into meaningful business insights using Python, Pandas, Plotly, and Streamlit.

# ⚙ Installation

Clone the repository:

```bash
git clone <your-github-repository-link>
```

Open the project:

```bash
cd Factory_Route_Efficiency
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# ▶️ Running the Project

Run the analysis:

```bash
python analysis.py
```

Launch the Streamlit dashboard:

```bash
streamlit run app.py
```
